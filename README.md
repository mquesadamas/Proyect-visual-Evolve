# Visual-Project-Evolve

## About the project
Visual Project Evolve es un proyecto de práctica enfocado en el análisis exploratorio de datos (EDA) sobre una tabla de partidos de fútbol obtenida de Kaggle. El dataset presentaba varios errores y problemas de estructura, por lo que fue necesario limpiar datos, reformular algunas columnas y crear nuevas variables para poder analizar mejor la información.

Aunque el contenido del dataset no sea especialmente complejo, el proyecto ha sido útil para practicar tareas fundamentales de análisis de datos, limpieza, transformación y visualización mediante tablas y gráficos.

## Objective
El objetivo principal del proyecto es estudiar cómo interactúan las variables entre sí y comprobar si algunas de ellas pueden tener relevancia a la hora de explicar o anticipar qué equipo gana un partido.

## Dataset
El conjunto de datos utilizado está basado en partidos de fútbol y procede de Kaggle.

Durante el trabajo con el dataset se realizaron tareas como:
- Limpieza de valores erróneos o poco consistentes.
- Reformulación de columnas existentes.
- Creación de nuevas variables derivadas.
- Preparación de los datos para su análisis visual y exploratorio.

## Tools and libraries
Este proyecto ha sido desarrollado en Python utilizando las siguientes librerías:
- pandas
- numpy
- matplotlib.pyplot
- seaborn

## Project workflow
El desarrollo del proyecto siguió una estructura de trabajo típica de EDA:
1. Carga y revisión inicial del dataset.
2. Detección de errores, columnas problemáticas y valores inconsistentes.
3. Limpieza y transformación de los datos.
4. Creación de nuevas variables para enriquecer el análisis.
5. Exploración visual de relaciones entre variables.
6. Interpretación de patrones relacionados con el resultado de los partidos.

## Project structure
- `data/`: archivos de datos
- `notebooks/`: análisis y visualizaciones
- `src/`: scripts principales
- `requirements.txt`: dependencias del proyecto
  
## Analysis
El análisis se centra en observar la relación entre distintas variables del dataset para identificar si existe alguna conexión relevante con el resultado final del partido.

A través de tablas y gráficos, el proyecto busca:
- Detectar patrones entre variables.
- Evaluar la posible relevancia de ciertos factores.
- Entender mejor qué variables pueden estar más asociadas con la victoria de un equipo.
- Practicar técnicas básicas de visualización y análisis exploratorio.
